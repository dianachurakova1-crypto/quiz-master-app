# Quiz Master - Веб-приложение для проведения квизов

## Описание
Веб-приложение для создания и проведения квизов в реальном времени с поддержкой комнат, WebSocket и автоматическим подсчётом баллов.

## Функционал
- Регистрация и авторизация пользователей
- Создание квизов с вопросами (одиночный/множественный выбор)
- Запуск квизов по коду комнаты
- Подключение участников в реальном времени (WebSocket)
- Автоматический подсчёт баллов
- Лидерборд и определение победителей
- История проведённых квизов

## Технологии
- Node.js + Express
- Socket.IO (WebSocket)
- HTML5, CSS3, JavaScript
- Хранение данных в памяти (MVP)
